## 🚀 快速创建 Agent

### 方式一：通过界面（推荐）

1. **访问模板市场**
   - 点击侧边栏"模板市场"

2. **选择模板**
   - 浏览模板卡片
   - 点击想要使用的模板

3. **配置参数**
   - 填写 Agent 名称
   - 选择 API 配置
   - 调整模板参数

4. **创建**
   - 点击"创建智能体"
   - 自动跳转到智能体列表

### 方式二：通过 API

```bash
# 1. 获取模板列表
curl http://localhost:8080/api/agent-templates

# 2. 查看模板详情
curl http://localhost:8080/api/agent-templates/simple_chat

# 3. 从模板创建 Agent
curl -X POST http://localhost:8080/api/agents/from-template \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "template_id": "simple_chat",
    "name": "我的助手",
    "api_config_id": 1,
    "params": {
      "personality": "friendly",
      "language_style": "balanced"
    }
  }'
```

## 📊 API 文档

### 获取模板列表
```
GET /api/agent-templates
Query: category (可选)
```

**响应：**
```json
{
  "code": 200,
  "data": {
    "templates": [...],
    "total": 5
  }
}
```

### 获取模板详情
```
GET /api/agent-templates/:id
```

### 获取分类列表
```
GET /api/agent-templates/categories
```

### 从模板创建 Agent
```
POST /api/agents/from-template
Authorization: Bearer TOKEN
Body: {
  "template_id": "simple_chat",
  "name": "Agent名称",
  "description": "描述",
  "api_config_id": 1,
  "is_public": false,
  "params": {...}
}
```

## 🔧 故障排查

### 问题1：模板列表为空
**解决方案：**
- 检查后端服务是否启动
- 查看浏览器控制台错误
- 确认 API 地址正确

### 问题2：创建失败
**可能原因：**
- API 配置未选择
- 参数验证失败
- Token 过期

**解决方案：**
- 检查必填字段
- 查看错误提示
- 重新登录

### 问题3：参数不生效
**解决方案：**
- 查看 Agent 的 system_prompt
- 确认参数正确传递
- 测试对话效果
