## 🚀 工作流能力对比

### 更新前可以做什么
```
简单工作流:
  Start → ChatModel → End

有限的工具调用:
  Start → ChatModel → Tool → End
```

### 更新后可以做什么
```
复杂的AI工作流:
  Start → ChatModel → Condition
    ├─ True → Tool → ChatModel → End
    └─ False → Lambda → End

数据处理管道:
  Start → HTTP Request → Lambda → Filter 
    → Transform → Loop → Merge → End

API集成系统:
  Start → Webhook → Switch
    ├─ Path A → HTTP Request → Template
    ├─ Path B → Lambda → ChatModel
    └─ Default → Delay → End

智能决策系统:
  Start → ChatModel → Condition → Loop
    → Tool → Transform → Merge → ChatModel → End
```
