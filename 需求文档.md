# AI智能体网站需求文档

## 项目概述

构建一个功能完善的AI智能体对话平台，支持多API接入、对话管理、智能体配置等功能。

---

## 一、核心功能需求

### 1.1 多API接入管理

#### 1.1.1 API配置系统
- **优先级1：OpenRouter.ai 集成**
  - 支持OpenRouter API密钥配置
  - 支持300+模型选择
  - 支持模型路由和故障转移
  - 支持成本追踪

- **优先级2：其他主流API支持**
  - OpenAI API
  - Anthropic Claude API
  - Google Gemini API
  - Azure OpenAI
  - 本地模型API（Ollama、LM Studio等）
  - 自定义API端点

#### 1.1.2 灵活的字段映射配置
- **请求字段映射**
  - API端点URL自定义
  - 请求头（Headers）自定义
  - 认证方式配置（Bearer Token、API Key等）
  - 消息格式映射（messages、prompt等）
  - 参数映射（temperature、max_tokens、top_p等）

- **响应字段映射**
  - 响应内容路径配置（如：choices[0].message.content）
  - Token使用量字段映射
  - 错误信息字段映射
  - 流式响应格式配置

- **预设模板**
  - OpenAI格式模板
  - Claude格式模板
  - Gemini格式模板
  - 自定义模板保存与复用

### 1.2 对话持久化存储（MySQL）

#### 1.2.1 数据库设计
- **用户表（users）**
  - 用户ID、用户名、邮箱、密码
  - 创建时间、最后登录时间
  - 用户偏好设置

- **智能体表（agents）**
  - 智能体ID、名称、描述、图标
  - 系统提示词
  - 模型配置（API类型、模型名称、参数）
  - 创建者ID、是否公开
  - 创建/更新时间

- **对话会话表（conversations）**
  - 会话ID、标题、智能体ID
  - 用户ID、创建时间、更新时间
  - 会话状态（活跃、归档、删除）
  - 总Token使用量

- **消息表（messages）**
  - 消息ID、会话ID
  - 角色（user、assistant、system）
  - 内容、附件（图片、文件）
  - Token使用量（输入/输出）
  - 创建时间、编辑状态

- **API配置表（api_configs）**
  - 配置ID、用户ID
  - API类型、名称
  - 端点URL、认证信息（加密存储）
  - 字段映射JSON
  - 是否启用

- **Token使用统计表（token_usage）**
  - 记录ID、用户ID、智能体ID
  - 日期、输入Token、输出Token
  - 成本估算

### 1.3 提示词管理

#### 1.3.1 系统提示词
- **自定义系统提示词**
  - 富文本编辑器
  - 变量插入（{user_name}、{date}等）
  - 提示词模板库
  - 提示词版本管理

#### 1.3.2 提示词模板
- **预设模板分类**
  - 编程助手
  - 写作助手
  - 翻译专家
  - 数据分析
  - 创意设计
  - 客服助手

- **用户自定义模板**
  - 创建、编辑、删除
  - 模板分享与导入
  - 模板评分与收藏

### 1.4 对话管理

#### 1.4.1 基础功能
- **新建对话**
  - 选择智能体
  - 自动/手动命名
  - 对话分组/标签

- **删除对话**
  - 单个删除
  - 批量删除
  - 软删除与彻底删除

#### 1.4.2 消息操作

#### 1.5.3 配额管理

### 1.6 MCP工具支持

#### 1.6.1 MCP协议集成

#### 1.6.2 内置工具

## 五、数据库设计详细说明

### 5.1 核心表结构

```sql
-- 用户表
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    avatar_url VARCHAR(500),
    role ENUM('free', 'premium', 'enterprise', 'admin') DEFAULT 'free',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    last_login_at TIMESTAMP,
    INDEX idx_email (email),
    INDEX idx_username (username)
);

-- API配置表
CREATE TABLE api_configs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    api_type VARCHAR(50) NOT NULL,
    endpoint_url VARCHAR(500) NOT NULL,
    auth_type ENUM('bearer', 'api_key', 'custom') DEFAULT 'bearer',
    credentials TEXT NOT NULL, -- 加密存储
    field_mapping JSON, -- 字段映射配置
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    INDEX idx_user_id (user_id)
);

-- 智能体表
CREATE TABLE agents (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    avatar_url VARCHAR(500),
    system_prompt TEXT,
    api_config_id INT,
    model_name VARCHAR(100),
    model_params JSON, -- temperature, max_tokens等
    tools JSON, -- MCP工具配置
    is_public BOOLEAN DEFAULT FALSE,
    usage_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (api_config_id) REFERENCES api_configs(id) ON DELETE SET NULL,
    INDEX idx_user_id (user_id),
    INDEX idx_public (is_public),
    FULLTEXT idx_search (name, description)
);

-- 对话会话表
CREATE TABLE conversations (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    agent_id INT NOT NULL,
    title VARCHAR(200),
    status ENUM('active', 'archived', 'deleted') DEFAULT 'active',
    total_tokens INT DEFAULT 0,
    total_cost DECIMAL(10, 6) DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    deleted_at TIMESTAMP NULL,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE CASCADE,
    INDEX idx_user_id (user_id),
    INDEX idx_agent_id (agent_id),
    INDEX idx_status (status),
    INDEX idx_updated_at (updated_at)
);

-- 消息表
CREATE TABLE messages (
    id INT PRIMARY KEY AUTO_INCREMENT,
    conversation_id INT NOT NULL,
    role ENUM('user', 'assistant', 'system') NOT NULL,
    content TEXT NOT NULL,
    attachments JSON, -- 图片、文件等
    input_tokens INT DEFAULT 0,
    output_tokens INT DEFAULT 0,
    metadata JSON, -- 工具调用、引用等额外信息
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (conversation_id) REFERENCES conversations(id) ON DELETE CASCADE,
    INDEX idx_conversation_id (conversation_id),
    INDEX idx_created_at (created_at)
);

-- Token使用统计表
CREATE TABLE token_usage (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    agent_id INT,
    conversation_id INT,
    date DATE NOT NULL,
    input_tokens INT DEFAULT 0,
    output_tokens INT DEFAULT 0,
    estimated_cost DECIMAL(10, 6) DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE SET NULL,
    FOREIGN KEY (conversation_id) REFERENCES conversations(id) ON DELETE SET NULL,
    UNIQUE KEY unique_daily_usage (user_id, agent_id, date),
    INDEX idx_user_date (user_id, date),
    INDEX idx_agent_date (agent_id, date)
);

-- 提示词模板表
CREATE TABLE prompt_templates (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category VARCHAR(50),
    content TEXT NOT NULL,
    variables JSON, -- 模板变量定义
    is_public BOOLEAN DEFAULT FALSE,
    usage_count INT DEFAULT 0,
    rating DECIMAL(3, 2) DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    INDEX idx_category (category),
    INDEX idx_public (is_public),
    FULLTEXT idx_search (name, description)
);

-- MCP工具表
CREATE TABLE mcp_tools (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category VARCHAR(50),
    config JSON NOT NULL, -- 工具配置
    is_builtin BOOLEAN DEFAULT FALSE,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_category (category),
    INDEX idx_active (is_active)
);
```

## 六、API设计规范

### 6.1 RESTful API端点

#### 用户相关
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出
- `GET /api/users/profile` - 获取用户信息
- `PUT /api/users/profile` - 更新用户信息

#### API配置
- `GET /api/configs` - 获取API配置列表
- `POST /api/configs` - 创建API配置
- `PUT /api/configs/:id` - 更新API配置
- `DELETE /api/configs/:id` - 删除API配置
- `POST /api/configs/:id/test` - 测试API连接

#### 智能体管理
- `GET /api/agents` - 获取智能体列表
- `POST /api/agents` - 创建智能体
- `GET /api/agents/:id` - 获取智能体详情
- `PUT /api/agents/:id` - 更新智能体
- `DELETE /api/agents/:id` - 删除智能体
- `GET /api/agents/public` - 获取公共智能体

#### 对话管理
- `GET /api/conversations` - 获取对话列表
- `POST /api/conversations` - 创建新对话
- `GET /api/conversations/:id` - 获取对话详情
- `PUT /api/conversations/:id` - 更新对话信息
- `DELETE /api/conversations/:id` - 删除对话
- `GET /api/conversations/:id/messages` - 获取对话消息

#### 消息处理
- `POST /api/conversations/:id/messages` - 发送消息
- `POST /api/conversations/:id/stream` - 流式对话（SSE）
- `DELETE /api/messages/:id` - 删除消息
- `PUT /api/messages/:id` - 编辑消息

#### Token统计
- `GET /api/usage/stats` - 获取使用统计
- `GET /api/usage/daily` - 获取每日统计
- `GET /api/usage/by-agent` - 按智能体统计

#### 提示词模板
- `GET /api/templates` - 获取模板列表
- `POST /api/templates` - 创建模板
- `GET /api/templates/:id` - 获取模板详情
- `PUT /api/templates/:id` - 更新模板
- `DELETE /api/templates/:id` - 删除模板

#### MCP工具
- `GET /api/tools` - 获取工具列表
- `POST /api/tools/:id/execute` - 执行工具
- `GET /api/tools/categories` - 获取工具分类
